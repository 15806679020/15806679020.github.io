一、Spring5.X注解配置

  • 开启注解配置和包扫描
    //主类
     public static void main(String[] args) {
        AnnotationConfigApplicationContext context = new AnnotationConfigApplicationContext();
        //扫描指定的包，包括子包
        context.scan("club.ldclass");
        //完成初始化操作，核心方法
        context.refresh();

        VideoService videoService = (VideoService) context.getBean("videoService");
        videoService.findById(44);
    }

二、@Configuration和@Bean

  • @Configuration标注在类上，相当于把该类作为Spring的xml配置文件

    • 作用为：配置Spring容器(应用上下文)

  • @bean注解：用于告诉方法产生一个Bean对象，然后这个Bean对象交给Spring管理，Spring将会将这个Bean对象放在自己的IOC容器中

  • 注意点:SpringIOC容器管理一个或者多个bean，这些bean都需要在@Configuration注解下进行创建

  • 实例

@Configuration
public class AppConfig {

    /**
     * 使用@Bean注解，表明这个bean交给spring管理
     * 如果没有制定名称，则默认采用 方法名+第一个字母小写 作为bean的名称
     * @return
     */
    @Bean(name = "vo",initMethod = "init",destroyMethod = "destroy")
    public VideoOrder videoOrder(){
        return  new VideoOrder();
    }
}
//videoOrder实体类
public class VideoOrder {
    private int id;
    private void init(){
        System.out.println("videoOrder init被调用");
    }
    private void destroy(){
        System.out.println("videoOrder destroy被调用");
    }
    public int getId() {
        return id;
    }
    public void setId(int id) {
        this.id = id;
        System.out.println("VideoOrder注入id时，调用setId方法");
    }
}
//主类
public static void main(String[] args) {
       AnnotationConfigApplicationContext context = new AnnotationConfigApplicationContext();
        //扫描指定的包，包括子包
        context.scan("club.ldclass");
        //完成初始化操作，核心方法
        context.refresh();
        VideoOrder videoOrder = (VideoOrder) context.getBean("vo");
    }

三、@PropertySource

  • @PropertySource，指定加载配置文件

    • 配置文件映射到实体类

  • 使用@Value映射到具体的java属性

  • 实例

    • 创建config.properties
    server.host=127.0.0.1
    • 创建CustomConfig.java
    @Configuration
@PropertySource(value = {"classpath:config.properties"})
public class CustomConfig {

    @Value("${server.host}")
    private String host;

    public String getHost() {
        return host;
    }

    public void setHost(String host) {
        this.host = host;
    }
}
    • 实现类
    @Service("VideoServiceImpl")
public class VideoServiceImpl implements VideoService {
    @Autowired
    private CustomConfig customConfig;

     @Override
    public Video findById(int id) {
        System.out.println("VideoServiceImpl 根据id查找视频");
        System.out.println(customConfig.getHost());
        return new Video();
    }
}
    • 测试
     public static void main(String[] args) {
      AnnotationConfigApplicationContext context = new AnnotationConfigApplicationContext();
        //扫描指定的包，包括子包
        context.scan("club.ldclass");
        //完成初始化操作，核心方法
        context.refresh();
        VideoService videoService = (VideoService) context.getBean("VideoServiceImpl");
        videoService.findById(44);
    }

四、注解配置AOP面向切面编程

  • 声明切面类 @Aspect(切面): 通常是一个类，里面可以定义切入点和通知，并配置切入点和通知
@Component
/**
 * 表明是一个切面类,此类定义切入点和通知
 */
@Aspect
public class LogAdvice {

    /**
     * 切入点
     */
    @Pointcut("execution(* club.ldclass.spring.service.VideoService.*(..))")
    public void aspect(){}

    /**
     * 前置通知
     * @param joinPoint
     */
    @Before("aspect()")
    public void beforeLog(JoinPoint joinPoint){
        System.out.println("LogAdvice beforeLog");
    }

    /**
     * 后置通知
     * @param joinPoint
     */
    @After("aspect()")
    public void afterLog(JoinPoint joinPoint){
        System.out.println("LogAdvice afterLog");
    }

    /**
     * 环绕通知
     */
    @Around("aspect()")
    public void around(JoinPoint joinPoint){
         System.out.println("LogAdvice around");
    }
}
  • 开启SpringAOP注解配置
    • 定义类AnnotationAppConfig.java
    • 主类中更改为 AnnotationConfigApplicationContext context = new AnnotationConfigApplicationContext(AnnotationAppConfig.class);
@Configuration
@ComponentScan("club.ldclass")
/**
 * 开启切面
 */
@EnableAspectJAutoProxy
public class AnnotationAppConfig {
}
  • 测试
public static void main(String[] args) {
        //定义注解配置类`AnnotationAppConfig.java`后，此一句话即可代替之前的
        AnnotationConfigApplicationContext context = new AnnotationConfigApplicationContext(AnnotationAppConfig.class);
        VideoService videoService = (VideoService) context.getBean("VideoServiceImpl");
        videoService.findById(44);
}

通过AOP的环绕通知统计方法调用耗时

  • 切面类LogAdvice
    • @Around
    /**
     * 环绕通知
     */
    @Around("aspect()")
    public void around(JoinPoint joinPoint){
        Object o = joinPoint.getTarget().getClass().getName();
        System.out.println("调用者="+o);
        //调用方法
        Object method = joinPoint.getSignature();
        System.out.println("调用方法="+method);
        //通过joinPoint获取参数
        Object[] args = joinPoint.getArgs();
        System.out.println("调用参数="+args[0]);
        long start = System.currentTimeMillis();
        System.out.println("环绕通知，环绕前===========");
        try {
            ((ProceedingJoinPoint)joinPoint).proceed();
        } catch (Throwable throwable) {
            throwable.printStackTrace();
        }
        long end = System.currentTimeMillis();
        System.out.println("环绕通知，环绕后===========");
        System.out.println("总耗时="+(end-start)+"ms");
    }
  • VideoServiceImpl
    • findById()
@Service("VideoServiceImpl")
